# Finger_Paint_Lua
A simple FingerPaint App, in Lua with Solar 2d

How to use:

Install <a href="https://docs.coronalabs.com/guide/start/installWin/index.html">Solar 2d</a>

Open main.lua in Solar2d Simulator
